¿Qué aprendimos?
- Más sobre CSS
- Unidades de medida absolutas y relativas
- Colores
- páginas: googleFonts, HTML Codes y Fontawesome

Proyecto: Consiste en una página HTML con estilos CSS para el diseño y la presentación de títulos de diferentes niveles. Se utilizan fuentes personalizadas y se aplican estilos de color y tamaño a los elementos de título. Además, se incorpora el uso de iconos de Font Awesome para agregar elementos visuales a los títulos. La página tiene un fondo de imagen de gatitos.
